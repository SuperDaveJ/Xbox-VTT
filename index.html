Welcome to the Xbox 360 Virtual Task Trainer (VTT)

Click the questions below to learn more about this VTT.

How can I use the VTT?

Navigate between sections of the VTT using the tabs. Detailed visual guidance for 3D parts navigation of this VTT is always available from the HELP menu.

The VTT is set up in four sections:

  • Explore - The explore tab allows you to spin, hold and see all Xbox 360 consoles and accessories.
  • Observe - The observe tab allows step by step observations of a Procedural Animations while walking through the steps from start to finish. Click each step to go back to a particular step and start again.
  • Hardware Practice - The Hardware Practice tab has Virtual Practice Exercises (VPE) where you can follow a step table but manipulate the models themselves to complete the step.
  • LIVE Practice - The LIVE Practice tab has Virtual Practice Exercises (VPE) where you can follow a step table but manipulate the models themselves to complete the step.

How can I succeed in the VTT?

When you complete the VTT, you are able to test your knowledge on the Virtual practice Exercises by logging on to the LMS. You must receive 80% on each Test to pass.

What will I learn in this VTT?

Upon completion, you should be able to:

  • Setup the original Xbox 360, Kinect and Accessories
  • Setup the Xbox 360S, Kinect and Accessories
  • Solve Xbox LIVE issues related to: IP Failure, Wireless Network Error, DNS Error and NAT Error
